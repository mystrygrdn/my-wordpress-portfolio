## 📖 About This Project

This project is a **Personal Web Portfolio** developed as the final requirement for the **Free Camp WordPress Development by Special Skills**. The website serves as a digital identity to showcase my skills as a Front-End Developer and UI/UX enthusiast.

The goal of this project was to create a high-performance, responsive, and visually appealing portfolio using industry-standard tools. It features a clean landing page, service highlights, a testimonial section with smooth anchor navigation, and a direct contact integration.

---

## ✨ Features

### 🎨 UI/UX
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile devices.
- **Custom Branding:** Features a minimalist logo and consistent color palette.
- **Smooth Navigation:** Implemented "Smooth Scroll" anchor links for a better user experience.

### ⚙️ Technical
- **Elementor Page Builder:** Custom layouts built from scratch without pre-made themes.
- **Dynamic Content:** Interactive sections including Testimonials and Portfolio Gallery.
- **Sticky Header:** Persistent navigation for easy access to site sections.
- **Version Control:** Connected via Git to track development progress from Pantheon to GitHub.
